# 學習環境
## Google Colab
* [https://colab.research.google.com/](https://colab.research.google.com/)
* 支援Python3環境
* 以Jupyter Notebook為基礎建造
* 在Google雲端運行
* 與Google Drive整合
* 無需安裝

### 初次運行設定
設定tab鍵縮排4格space  
`Settings` => `Editor` => `Indentation idth in spaces` => 4

顯示行號  
`Settings` => `Editor` => 
✅ Show line numbers 

常用快捷鍵
```
# 執行
Shift + Enter
```
